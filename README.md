# Tic Tac Toe – Solo or Duo

A responsive Tic Tac Toe game with **Two Player** mode and **Single Player**.

## Features
- Choose between Two Player or vs Computer
- Customizable player names
- Smart AI (wins, blocks, takes center/corners/sides)
- Score tracking and restart/reset options
- Animations on win, draw, and invalid moves
- Fully responsive (works on mobile)

## How to play
1. Open `index.html` in any modern browser.
2. Select game mode.
3. Enter names (optional).
4. Click on any cell to make a move.
5. In Single Player mode, you are **X** and the computer is **O**.
